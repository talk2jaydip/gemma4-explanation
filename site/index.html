The short version

Gemma 4 is a family, not one model. Four public sizes: E2B, E4B, 26B A4B, 31B. The edge models (E2B/E4B) add audio. The 26B A4B is the sparse MoE member - 25.2B stored parameters, ~3.8B active per token. Hybrid attention is what keeps the 256K context window tractable - most layers read only a window, a few read the full context, and the final layer is always global.

Honest starting point: pick the smallest model that preserves your needed modality, context length, and visual detail. Move up only when your eval set shows capability failures, not prompting failures.
